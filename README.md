# OptiCrop — Complete Project

## Setup
```
pip install -r requirements.txt
```
Place `Crop_recommendation.csv` inside the `data/` folder.

---

## Run Order

### Epic 2: Data Collection and Analysis
```bash
python epic2_data_collection_analysis.py
# Covers: Import Libraries, Read Dataset, Univariate, Bivariate, Multivariate Analysis
# Saves:  univariate_analysis.png, bivariate_analysis.png, multivariate_analysis.png
```

### Epic 3: Data Pre-processing
```bash
python step1_null_check.py          # Story 1: Null values
python step2_outliers.py            # Story 2: Outliers + boxplot_outliers.png
python step3_seasonal_crops.py      # Story 3: Seasonal crops
python step4_train_test_split.py    # Story 4: Train/test split
```

### Epic 4: Model Building
```bash
python step5_kmeans_elbow.py        # Story 1: K-Means + elbow_plot.png
python step6_logistic_regression.py # Story 2: Logistic Regression
python step7_evaluate_save.py       # Story 3: Evaluate + saves model.pkl & scaler.pkl
python step8_predict_crop.py        # Story 4: Predict crop (run AFTER step7)
```

### Epic 5: Application Building
```bash
python app.py   # Run AFTER step7. Open http://127.0.0.1:5000
```
